# FLOW Framework Skill

## Overview

Strategic SEO planning using the FLOW framework: **F**ind, **L**everage, **O**ptimize, **W**in.

## Commands

| Command | Description |
|---------|-------------|
| `/seo flow find <url>` | Find opportunities |
| `/seo flow leverage <url>` | Leverage existing assets |
| `/seo flow optimize <url>` | Optimize current performance |
| `/seo flow win <url>` | Win competitive battles |
| `/seo flow full <url>` | Complete FLOW analysis |

## FLOW Stages

### Find (Discovery)
- Keyword research
- Content gap analysis
- Trending topics
- SERP opportunity mapping

### Leverage (Assets)
- Existing content audit
- Backlink opportunities
- Brand mentions
- Internal assets

### Optimize (Performance)
- Technical SEO fixes
- Content improvements
- UX enhancements
- Speed optimization

### Win (Competition)
- Competitive positioning
- SERP feature targeting
- Authority building
- Conversion optimization
